# Codebook for Coursera Getting Data Course Project

## Introduction

This is a codebook for the Coursera Getting Data Course Project. In this codebook we describe

1. The dataset we generate 
2. The transformations that we undertook
3. The variables in our dataset

We also provide 

1. Links to download the original dataset we used
2. Links to details about the original dataset

## Original Dataset

This dataset is derived from the "Human Activity Recognition Using Smartphones Data Set" which was originally made available at [http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones](http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones)

Our data was downloaded [here](https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip).

## Dataset Generated

We generate a tidy data set, per the guidelines set out by [Hadley Wickham](http://vita.had.co.nz/papers/tidy-data.pdf). Our data set consists of a set of averaged measurements for activity and subject combinations.

## Transformations

To generate this dataset we

1. Load in the source data files combined these into a single data frame
2. Add descriptive activity names to our dataframe
3. Clean up the names of our existing columns for readability
4. Create our tidy dataset by melting and recasting our combined dataframe, getting the mean of each activy and subjectid along the way.
